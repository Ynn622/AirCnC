🚘 AirCnC 使用者操作手冊

歡迎使用 AirCnC！ 這是一個專為汽機車共享打造的區塊鏈平台，您可以輕鬆將自己的車輛出租，或快速租借別人的車輛。以下是詳細的操作流程說明，讓您快速上手！

🔐 一、開始使用前

✅ 需要準備：

  • 一個支援區塊鏈的錢包（建議使用 MetaMask）
  • 一些測試用的 Sepolia ETH (可從測試水龍頭取得)
  • 使用桌面版 Chrome 瀏覽器，體驗更佳（也可使用MetaMask APP 開啟網頁）
  • 請確保您的瀏覽器已安裝 MetaMask 擴充功能、並連接到 Sepolia 測試網絡
👤 二、登入區塊鏈錢包
  1. 點擊網頁右上角的 「個人頭像」 按鈕
  2. 允許網站連接您的錢包帳戶
  3. 成功後會顯示您的錢包地址（0x 開頭）
🚗 三、車主功能操作說明

上架車輛

  1. 點選上方選單中的「上架車輛」
  2. 填寫車輛資訊：
    • 車型名稱
    • 車輛描述
    • 租金（每次租借所需費用）
  3. 點擊「上架」按鈕
  4. 確認錢包中的交易，等待鏈上確認
  5. 成功後，您的車輛將出現在「機車租借 / 汽車租借」頁面中
🚙 四、租客功能操作說明

查看可租車輛

  1. 進入首頁或「租車」頁面
  2. 瀏覽列表中的車輛資訊（如名稱、價格）
  3. 點擊「預約租車」按鈕選擇您想要的車輛

租賃車輛流程

  1. 點擊「確認預約」後，系統會請您確認交易
  2. 錢包跳出視窗後，點選「確認」
  3. 等待區塊鏈完成交易
  4. 租賃成功後，您可於個人頁面查看租借紀錄
📋 五、個人資料與管理

查看我的車輛

  1. 點擊網頁右上角的 「個人頭像」 按鈕
  2. 查看您已上架的車輛與租賃狀態
  3. 可進行「下架車輛」或「取消租約」
🔄 六、被預約後流程

交付與還車流程

  1. 請預約租車人聯繫車主，確認交付機車與鑰匙時間
  2. 當雙方見面並交付完成後，請雙方進入個人頁面，按下「確認租車」按鈕，即可開始租車
  3. 還車時請車主檢查車輛後，先按下「確認還車」
  4. 若有超時則需額外支付費用
  5. 當租車人確認還車後，車主也請按下「確認還車」，即可收款！
請注意：雙方都確認還車後，系統才會自動完成交易並將款項轉至車主帳戶。
❓ 常見問題
Q：我需要實際付款嗎？
A：本平台目前運作在測試鏈上，您可以使用測試幣進行操作，不會有實際金錢損失。
Q：我的交易卡住了怎麼辦？
A：請確認您的網路狀態與錢包是否正常連接，或稍候重新操作。
Q：為什麼看不到我剛剛的車輛？
A：區塊鏈需要時間確認交易，請耐心等候約 10–30 秒後重新整理頁面。
